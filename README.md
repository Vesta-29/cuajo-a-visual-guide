# Cuajo: A Digital Manual

**Cuajo: A Digital Manual** is a beautiful, interactive visual guide designed for beginners looking to learn the heritage Filipino card game of Cuajo. Originating from Spain and deeply embedded in the history of the Philippines, Cuajo is a fascinating 4-player rummy-style game played with a unique 112-card deck.

This application breaks down the complex rules into easily digestible, highly visual sections:

- **The Deck**: Understanding the four suits (Oros, Copas, Espadas, Bastos) and the unique card composition.
- **The Objective**: Building a 16-card winning hand (Bahay).
- **Gameplay**: Learning the deal, draw, discard mechanics, and special rules like _Sowee_ and _Bunot_.
- **Scoring**: A comprehensive visual breakdown of winning combinations, bonus payouts, and super rare win conditions like _Porbis_.

## Key Features

- **Interactive Scenarios**: Hover over scenario diagrams to see step-by-step scoring breakdowns connecting cards dynamically.
- **Multiple Themes**: Swap between Classic, Faded (Vintage Sepia), and Night mode themes to read the manual comfortably in any environment.
- **Responsive Layout**: Built with Tailwind CSS to look premium and legible on both mobile devices and large desktop monitors.

## Run Locally

**Prerequisites:** Node.js

1. Clone this repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open the provided `localhost` URL in your browser.
